# Compliance Shield Project

This repository contains the source code for the Compliance Shield web application, a multi-agent regulatory audit system tailored for financial document governance in the Indian legal and financial landscape.

## Getting Started

To run the project locally, ensure you have Node.js and npm installed.

```sh
# Clone the repository
mkdir compliance-shield && cd compliance-shield
# assuming the repo is already checked out in this path

# Install dependencies
npm install

# Start the development server
npm run dev
```

Feel free to make changes using your preferred editor. Pushed changes will update the live application if deployed.

## Technologies Used

- Vite
- React (with TypeScript)
- Tailwind CSS
- shadcn-ui components
- Other libraries listed in `package.json`

## Deployment

Build the production bundle with:

```sh
npm run build
```

then serve the `dist` directory using your hosting provider of choice.
